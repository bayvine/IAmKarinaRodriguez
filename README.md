# I Am Karina Rodriguez

Next.js 16 starter with Tailwind CSS v4, Prismic Slice Machine, Radix UI
primitives, `clsx` + `tailwind-merge`, and Framer Motion.

## Included

- App Router setup
- Tailwind CSS v4 theme tokens and utility-friendly class composition
- Prismic client, preview routes, and Slice Simulator
- Radix dropdown primitive starter
- Motion-ready reveal component
- Font slots for Inter and Gambarino

## Commands

```bash
npm install
npm run dev
npm run slicemachine
npm run lint
npm run typecheck
```

## Fonts

The project is currently wired to these local files:

- `public/fonts/inter/Inter_18pt-*.ttf`
- `public/fonts/gambarino/Gambarino-Regular.otf`

## Prismic

The repository is already set in `slicemachine.config.json` as
`iamkarinarodriguez`.

Update the route resolver array in `prismicio.ts` once your page types are
finalized.
